## How to Play

1. Open `index.html` in your web browser
2. Click "Start Game" to begin
3. **Controls**:
   - Press `SPACE` or `CLICK` to make the bird flap
   - Press `ESC` to pause the game
4. Navigate through the pipes to score points
5. Avoid hitting pipes and boundaries

## Game Controls

| Control | Action |
|---------|--------|
| SPACE   | Flap/Jump |
| CLICK   | Flap/Jump |
| ESC     | Pause Game |

## Settings

Configure the game in the Settings menu:
- **Difficulty**: Adjust game speed and challenges
- **Background Music**: Toggle different music tracks
- **Sound Volume**: Control overall game volume
